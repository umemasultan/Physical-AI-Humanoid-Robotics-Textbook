# Physical AI & Humanoid Robotics

**Essentials for the AI-Native Era**

Welcome to this comprehensive textbook on Physical AI and Humanoid Robotics. This course will take you from the fundamentals of embodied intelligence to building your own AI-robot pipeline.

## What You'll Learn

This textbook covers six key areas:

1. **[Introduction to Physical AI](/intro-physical-ai)** — Understanding embodied intelligence and its applications
2. **[Basics of Humanoid Robotics](/humanoid-basics)** — Robot anatomy, kinematics, and URDF
3. **[ROS 2 Fundamentals](/ros2-fundamentals)** — The Robot Operating System for modern robotics
4. **[Digital Twin Simulation](/digital-twin)** — Gazebo, Isaac Sim, and virtual testing
5. **[Vision-Language-Action Systems](/vla-systems)** — Multimodal AI for robotics
6. **[Capstone Project](/capstone)** — Build a complete voice-commanded robot pipeline

## How to Use This Textbook

- **Read sequentially** — Chapters build on each other
- **Try the exercises** — Hands-on practice is essential
- **Use the AI chatbot** — Ask questions about any topic (coming soon)
- **Check the glossary** — Reference key terms anytime

## Prerequisites

- Basic programming experience (Python preferred)
- Familiarity with command line interfaces
- Curiosity about robotics and AI!
